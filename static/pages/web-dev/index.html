Activité - Developpement Web

Nom

Activité de Developpement Web

Description

L’activité parascolaire de développement web offre aux étudiants l’opportunité de découvrir et de maîtriser les compétences essentielles pour créer des sites web interactifs et fonctionnels. Les participants apprendront les bases du HTML, CSS, et JavaScript. Cette activité est idéale pour les élèves passionnés par la technologie et la création numérique.
